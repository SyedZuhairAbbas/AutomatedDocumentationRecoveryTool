Capabilities

What DocGen Extracts

⚙️
Function Detection
Automatically identifies every standalone function in the codebase, including its name and exact line number.
🏛
Class Analysis
Detects class definitions and maps all their methods, constructors, and internal structure.
📥
Parameter Extraction
Lists all parameters for every function and method, automatically excluding the self reference.
📝
Docstring Recovery
Recovers any existing docstrings embedded in the code. Generates smart descriptions when none exist.
📍
Line Mapping
Every extracted element is mapped to its exact line number in the original source file for easy navigation.
💾
Markdown Export
One-click export of all generated documentation into a clean, shareable Markdown file.
Architecture

Tech Stack

FRONTEND

JavaFX

Cross-platform desktop UI framework. Handles file picking, result display, and Markdown export with a clean dark themed interface.

BACKEND

Python FastAPI

High-performance REST API server. Receives uploaded files, processes them, and returns structured JSON responses.

ANALYSIS

Python AST Module

Built-in Abstract Syntax Tree parser. Converts raw source code into a navigable tree structure without executing any code.

PARSING

Gson + HttpClient

Java's built-in HttpClient handles multipart file uploads. Gson parses the JSON response into structured documentation output.
